
Time Series Lab

Setup:

Mac/Linux:
chmod +x install_env.sh
./install_env.sh
source .venv/bin/activate

Windows (PowerShell):
.\install_env.ps1
venv\Scripts\Activate.ps1

Then in VS Code:
1. Ctrl/Cmd+Shift+P
2. Python: Select Interpreter
3. Choose .venv
4. Ctrl/Cmd+Shift+P
5. Developer: Reload Window

Open notebook/notebook.ipynb
